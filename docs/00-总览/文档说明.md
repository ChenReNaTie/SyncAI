# SyncAI 文档说明

**更新时间：** 2026-04-30

## 1. 当前正式文档规则

从当前版本开始，`docs/` 下的正式文档统一使用中文文件名，并按主题放入子目录管理。

执行原则：

- 正式文档优先保留当前有效结论，不重复保留已被取代的英文旧稿
- 历史过程材料仅保留必要中文归档，不再继续扩散英文草稿
- 新增文档时，先判断应归入哪个主题目录，再决定是否需要单独成文

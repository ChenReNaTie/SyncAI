# SyncAI（灵悉 AI）使用说明

## 概述

SyncAI 是一个 AI 协作平台，核心概念层次：

```
团队（Team） → 项目（Project） → 会话（Session） → 消息（Message）
```

每个层级都是父子关系：一个团队里可以有多个项目，一个项目里可以有多个会话，一个会话里包含多轮消息。

---

## 1. 注册和登录

### 注册账号

1. 浏览器打开 `http://localhost:5173`
2. 没有登录时，会自动跳转到 **登录/注册页面**
3. 点「**去注册**」切换到注册模式
4. 填写以下信息：
   - **邮箱**：用于登录的邮箱地址
   - **密码**：至少 8 位
   - **显示名**：你希望在团队中显示的名字
5. 点击「**注册**」按钮
6. 注册成功后**自动登录**，直接进入 Dashboard

### 登录已有账号

1. 在登录页直接输入 **邮箱** 和 **密码**
2. 点击「**登录**」

> 💡 登录后 token 保存在浏览器 `localStorage` 中，刷新页面不会掉线。

---

## 2. Dashboard（工作台）

登录后首先进入 Dashboard，这里是你的**主控面板**，可以做三件事：

### 搜索历史消息

Dashboard 顶部是搜索区：

1. 在「**Search**」输入框输入关键词
2. 在下拉框中选择一个已加入的**团队**
3. 点击「**Search**」按钮
4. 搜索结果会显示匹配的消息内容、所属项目和会话、发送时间

> 搜索范围是整个团队内的所有会话消息。

### 查看我的团队

搜索区下方列出了你已加入的所有团队，每个团队卡片显示：
- 团队名称（点击可进入团队页面）
- 团队 Slug（唯一标识）
- 创建时间

### 创建新团队

Dashboard 底部是创建团队的表单：

1. **Team Name**：团队显示名称（如「我的产品团队」）
2. **Team Slug**：团队唯一标识（如 `my-product-team`，只允许小写字母、数字和连字符）
3. 点击「**Create Team**」

创建成功后团队列表自动刷新，新团队会出现在列表中。

---

## 3. 团队页

在 Dashboard 点击**团队名称**进入团队页。

### 创建项目

1. 点击「**+ Create Project**」按钮
2. 填写：
   - **Project Name**：项目名称（必填）
   - **Description**：项目描述（选填）
3. 点击「**Create Project**」

### 进入项目

在项目列表中点击**项目名称**即可进入项目页。

### 配置 Agent 节点

团队页后续可以配置 AI Agent 节点（具体方式取决于部署版本），包括：
- 添加/移除 AI 模型节点
- 配置 Agent 的角色和行为
- 设置会话路由规则

---

## 4. 项目页

在团队页点击**项目名称**进入项目页。

### 创建会话

1. 点击「**+ Create Session**」按钮
2. 填写：
   - **Session Title**：会话标题（必填）
   - **Visibility**：可见性（Shared 对团队公开 / Private 仅自己可见）
3. 点击「**Create Session**」

### 筛选会话

可以通过顶部的下拉菜单筛选会话：
- **All**：显示所有会话
- **Shared**：只显示共享会话
- **Private**：只显示私有会话

### 进入会话

在会话列表中点击**会话标题**即可进入会话页开始对话。

---

## 5. 会话页（核心功能）

会话页是 SyncAI 的核心交互界面，在这里与 AI 进行实时对话。

### 发送消息

1. 在页面底部的输入框中输入消息内容
2. 点击「**Send**」按钮（或按回车键）
3. 你的消息立即出现在消息列表中
4. AI 的回复会**实时**推送到消息列表中（通过 WebSocket）

> 页面顶部会显示当前的**运行时状态**（如 `idle`、`thinking` 等），帮你了解 AI 当前在做什么。

### Replay（时间轴回放）

回放功能让你按时间顺序查看会话的完整对话历史：

1. 点击页面顶部的「**Replay**」按钮
2. 所有消息会按**时间从早到晚**排列展示
3. 顶部出现 `[REPLAY]` 标记，输入框隐藏
4. 点击「**Exit Replay**」退出回放模式，回到实时对话

### 创建待办（Todo）

可以把 AI 回复中的某条消息转成待办：

1. 在消息右侧点击「**+ Todo**」按钮
2. 弹出一个输入框，填写待办标题
3. 按回车或点击「**Add**」确认

创建的 Todo 会出现在**会话底部的 Todo 列表中**。

### 管理待办

Todo 列表显示在消息列表下方：

- 每个 Todo 显示**状态标签**（`pending` / `done`）
- **标记完成**：点击「**Done**」按钮，Todo 状态变为 `done`，标题出现删除线
- **重新打开**：点击「**Reopen**」按钮，Todo 状态恢复为 `pending`

> Todo 数量会在标题中显示：`Todos (3 pending)` 告诉你还有多少个没完成。

### 实时状态

通过 WebSocket 连接，会话页会实时显示：
- **新消息推送**：AI 回复到达后自动加入消息列表
- **运行时状态变化**：标题旁显示 `[idle]`、`[thinking]` 等状态

---

## 6. 退出登录

在 **Dashboard 页面**点击右上角的「**Logout**」按钮即可退出登录。

退出后会清除本地 token，跳回登录页。

---

## 7. 页面导航速查

| 从 | 到 | 方式 |
|----|----|------|
| 登录页 | Dashboard | 登录/注册成功后自动跳转 |
| Dashboard | 团队页 | 点击团队名称 |
| 团队页 | Dashboard | 点击「← Back to Dashboard」 |
| 团队页 | 项目页 | 点击项目名称 |
| 项目页 | Dashboard | 点击「← Back to Dashboard」 |
| 项目页 | 会话页 | 点击会话标题 |
| 会话页 | 项目页 | 点击「← Back to Project」 |

---

## 8. 层次关系总结

```
Dashboard
├── 搜索历史消息（跨团队关键词搜索）
├── 创建团队
├── 查看团队列表
└── 退出登录

团队页（/teams/:teamId）
├── 配置 Agent 节点
├── 创建项目
└── 查看项目列表

项目页（/projects/:projectId）
├── 创建会话
├── 筛选会话
└── 查看会话列表

会话页（/sessions/:sessionId）
├── 发送消息（实时 AI 回复）
├── Replay 回放
├── 创建/管理 Todo
└── 查看运行时状态
```
